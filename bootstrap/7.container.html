Bootstrap Container Examples

1. .container
This container has a responsive fixed width (changes at breakpoints). Resize the window to see.
2. .container-fluid
This container is always full width (100% of the screen), regardless of screen size.
3. .container-sm
This container has 100% width until the "sm" breakpoint (576px), then becomes fixed width.
4. .container-md
This container has 100% width until the "md" breakpoint (768px), then becomes fixed width.
5. .container-lg
This container has 100% width until the "lg" breakpoint (992px), then becomes fixed width.
6. .container-xl
This container has 100% width until the "xl" breakpoint (1200px), then becomes fixed width.
7. .container-xxl
This container has 100% width until the "xxl" breakpoint (1400px), then becomes fixed width.
