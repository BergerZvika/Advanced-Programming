Learning CSS Units: vh, vw, and %

% - Percentage of Parent

Width = 50%

50% width means 50% of the width of its parent element (in this case, the section box). It depends on parent size.

vw - Viewport Width

Width = 50vw

50vw means the width is 50% of the browser's viewport width. Resize the window horizontally to see it change.

vh - Viewport Height

Height = 30vh

30vh means the height is 30% of the browser's viewport height (screen height), not of the parent element. Try resizing the window vertically!
