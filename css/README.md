# CSS Tutorial

A practical CSS tutorial organized into 5 topic folders, each with standalone HTML/CSS examples you can open directly in a browser.

## Structure

### 1. Selectors
Targeting elements with CSS selectors.

| File | Topic |
|------|-------|
| `basic selectors.html` | Element, class, ID, and attribute selectors |
| `child selector.html` | Child (`>`), descendant, sibling (`~`, `+`) selectors |
| `pseudo selectors.html` | Pseudo-classes (`:hover`, `:nth-child`) and pseudo-elements (`::before`, `::after`) |

### 2. Colors
Working with color in CSS.

| File | Topic |
|------|-------|
| `colors.html` | Named colors, hex, `rgb()`, `hsl()`, opacity |
| `colors.css` | Shared color variables and utilities |
| `gradients.html` | `linear-gradient`, `radial-gradient`, `conic-gradient` |

### 3. Size
Sizing and spacing units.

| File | Topic |
|------|-------|
| `basic size.html` | `px`, `%`, `em`, `rem` |
| `advanced-sizing.html` | `min()`, `max()`, `clamp()`, `fit-content` |
| `viewport.html` | `vw`, `vh`, `vmin`, `vmax`, `dvh` |
| `size.css` | Shared sizing utilities |

### 4. Page Decoration
Layout and visual styling.

| File | Topic |
|------|-------|
| `basic decoration.html` | Borders, shadows, border-radius, backgrounds |
| `flex.html` | Flexbox: `display:flex`, alignment, wrapping |
| `grid.html` | CSS Grid: `grid-template`, `gap`, `grid-area` |
| `fonts.html` | Font families, `@font-face`, Google Fonts, `font-display` |
| `z-index.html` | Stacking context and `z-index` |

### 5. Transition & Animation
Motion and interactivity.

| File | Topic |
|------|-------|
| `basic-transitions.html` | `transition` shorthand, timing functions |
| `more-transitions.html` | Multi-property transitions, chaining |
| `more hover.html` | Advanced hover effects |
| `proprety.html` | Animatable CSS properties reference |
| `darkMode.html` | Dark mode with `prefers-color-scheme` |

## Usage

Open any `.html` file directly in your browser — no build step required.
