Canvas Animation

Animation is done by repeatedly clearing and redrawing the canvas using requestAnimationFrame — the browser calls your draw function at 60 fps, synchronized to the display refresh.

function draw(timestamp) {
  ctx.clearRect(0, 0, canvas.width, canvas.height); // erase previous frame
  // ... draw current frame based on timestamp or state ...
  requestAnimationFrame(draw); // schedule next frame
}
requestAnimationFrame(draw); // kick it off
