SVG Animation

SVG can be animated in three ways: CSS animations/transitions, SMIL (built-in SVG animation elements), and JavaScript.

CSS Animations on SVG

Spin (rotate)

Pulse (scale)

Stroke Draw-on

Float

SMIL — Built-in SVG Animation

SMIL animations are declared directly in SVG markup using <animate>, <animateTransform>, and <animateMotion>.
