Inline SVG

SVG (Scalable Vector Graphics) is an XML-based format for vector drawings. Unlike <canvas>, SVG elements are in the DOM — you can style them with CSS and target them with JavaScript.

CSS & JS Integration

SVG elements inside HTML can be styled with CSS and targeted with JavaScript just like regular elements.

// CSS hover via stylesheet
#svg-circle:hover { fill: #e74c3c; }

// JavaScript interaction
circle.addEventListener('click', () => {
  circle.setAttribute('r', '60');
  circle.style.fill = '#f39c12';
});
