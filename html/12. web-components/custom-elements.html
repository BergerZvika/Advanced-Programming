Custom Elements

Custom elements let you define your own HTML tags with built-in behaviour — part of the Web Components standard.

Lifecycle Callbacks

CallbackWhen it fires
constructor()Element is created (don't touch the DOM here)
connectedCallback()Element is inserted into the document
disconnectedCallback()Element is removed from the document
attributeChangedCallback(name, oldVal, newVal)An observed attribute changes
static observedAttributesReturn an array of attribute names to watch
adoptedCallback()Element is moved to a new document (rare)

Defining Custom Elements

class MyElement extends HTMLElement {
  // Attributes to watch for changes
  static get observedAttributes() { return ['value', 'color']; }

  constructor() {
    super();
    // Attach a shadow root (encapsulated DOM + CSS)
    this.attachShadow({ mode: 'open' });
  }

  connectedCallback() {
    this.render();
  }

  attributeChangedCallback(name, oldVal, newVal) {
    if (oldVal !== newVal) this.render();
  }

  render() {
    const value = this.getAttribute('value') ?? 0;
    this.shadowRoot.innerHTML = `<span>Value: ${value}</span>`;
  }
}

// Register the tag name (must contain a hyphen)
customElements.define('my-element', MyElement);
