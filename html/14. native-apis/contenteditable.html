contenteditable

The contenteditable attribute makes any element directly editable by the user — the foundation of rich-text editors like Medium, Notion, and Quill.

Values

ValueEffect
contenteditable="true"Element and its children are editable
contenteditable="false"Explicitly not editable (useful inside a true parent)
contenteditable="plaintext-only"Editable but strips all formatting — plain text only

Important Notes

TopicNotes
AccessibilityAdd role="textbox" and aria-multiline="true" so screen readers announce the field correctly
execCommanddocument.execCommand() is deprecated but still works everywhere. Modern replacement: Selection API + insertText via InputEvent
XSS riskNever insert innerHTML from a contenteditable directly into the page without sanitization (DOMPurify)
PersistenceRead the content with element.innerHTML or element.innerText and POST it to your server
plaintext-onlyUse contenteditable="plaintext-only" to strip pasted HTML — good for simple input fields
