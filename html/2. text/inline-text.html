Inline Text Elements

These elements style text within a line without starting a new block.

ElementRenderedMeaning
<strong> Important text Important — bold by default; conveys semantic weight
<em> Emphasized text Stress emphasis — italic by default
<b> Bold text Visually bold — no semantic importance (use <strong> instead when meaning it)
<i> Italic text Visually italic — for technical terms, foreign words, titles
<u> Underlined text Non-link underline (e.g., spell-check annotation)
<s> Strikethrough No longer accurate/relevant (not for edits — use <del>)
<del> / <ins> removed added Document edits / tracked changes
<mark> Highlighted Highlighted / search result
<code> console.log() Inline code snippet (monospace)
<kbd> Ctrl + C Keyboard input
<abbr> HTML (hover me) Abbreviation with tooltip via title
<sup> / <sub> H2O   x2 Subscript / Superscript
<small> Fine print text Side comments, disclaimers, copyright
<q> Inline quotation Short inline quote (browser adds quotation marks)
<blockquote>
A longer quoted passage from another source.
 Block-level quotation; use cite attribute for source URL
