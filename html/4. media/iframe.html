<iframe> — Inline Frames

An iframe embeds another HTML document inside your page — used for maps, YouTube, payment widgets, and more.

Basic iframe

<iframe
  src="https://example.com"
  width="100%"
  height="250"
  title="Example Domain">
</iframe>

Key Attributes

AttributePurpose
srcURL of the document to embed
titleAccessible label (required for screen readers)
width / heightDimensions (pixels or %)
loading="lazy"Defer loading until near viewport
allowfullscreenAllow the content to go fullscreen
sandboxRestrict what the embedded page can do (see below)

sandbox Attribute

The sandbox attribute applies restrictions by default. Add tokens to re-enable capabilities:

<!-- fully sandboxed (scripts, forms, popups all blocked) -->
<iframe src="untrusted.html" sandbox></iframe>

<!-- allow scripts and same-origin only -->
<iframe src="widget.html" sandbox="allow-scripts allow-same-origin"></iframe>
TokenWhat it re-enables
allow-scriptsJavaScript execution
allow-formsForm submission
allow-same-originAccess to cookies/storage from same origin
allow-popupswindow.open() and target="_blank"
allow-top-navigationNavigate the parent page

Embedding a Google Map

<iframe
  src="https://maps.google.com/maps?q=Berlin&output=embed"
  width="600" height="400"
  title="Map of Berlin"
  loading="lazy"
  allowfullscreen>
</iframe>
