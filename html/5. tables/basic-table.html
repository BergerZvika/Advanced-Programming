Element Reference

TagPurpose
<table>Wrapper for the entire table
<caption>Title/description of the table (accessibility)
<thead>Header section — browsers may repeat on print page breaks
<tbody>Main data rows (can have multiple <tbody> groups)
<tfoot>Footer row (totals, summaries)
<tr>Table row
<th>Header cell — bold & centered by default; use scope for accessibility
<td>Data cell
