Advanced Form Validation

Combines HTML5 constraint attributes, :user-invalid / :user-valid CSS pseudo-classes, and the Constraint Validation API for full control.

Constraint Validation API

const input = document.getElementById('username');

// Check validity
input.checkValidity();       // true / false
input.reportValidity();      // same + shows browser tooltip

// ValidityState properties
input.validity.valueMissing  // true if required and empty
input.validity.typeMismatch  // true if email/url format wrong
input.validity.patternMismatch // true if pattern doesn't match
input.validity.tooShort      // true if below minlength
input.validity.tooLong       // true if above maxlength
input.validity.rangeUnderflow // true if below min
input.validity.rangeOverflow  // true if above max
input.validity.valid         // true only if all checks pass

// Custom error message
input.setCustomValidity('Username is taken');
input.setCustomValidity(''); // clear custom error
