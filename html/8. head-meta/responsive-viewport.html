Resize me!

I change with the viewport

Drag the editor/preview divider (or resize the window) to see this card switch colours at 600px and 900px.

The <meta name="viewport"> tag tells mobile browsers to render at the device's real width — without it, phones simulate a ~980px desktop and zoom out.

CSS @media (min-width: ...) queries branch styles based on the current viewport size. Mobile-first writes the default styles for the smallest screen and adds rules for wider ones.
